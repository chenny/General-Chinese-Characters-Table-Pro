# 通用规范汉字表 · 背景介绍

## 一、什么是《通用规范汉字表》？

《通用规范汉字表》是由中华人民共和国教育部、国家语言文字工作委员会于 **2013年6月5日** 正式发布的国家语言文字规范性标准文件，编号 **GB/T 13000**，收录汉字共 **8105 个**，是迄今最权威的现代汉语规范字表。

它是 1988 年发布的《现代汉语常用字表》（7000字）的全面升级版。

---

## 二、字表分级说明

| 级别 | 数量 | 核心用途 |
|:---:|:---:|:---|
| **一级字表** | **3500** 字 | 覆盖日常出版物（报纸、课本）99% 以上用字 |
| **二级字表** | **3000** 字 | 满足一般性阅读需求，补充一级字表 |
| **三级字表** | **1605** 字 | 专门领域：人名、地名、科技术语、文言文用字 |

> **总计 8105 字**，Unicode 全覆盖，满足现代信息处理所有主流场景。

---

## 三、数据字段说明

本仓库数据来源于国家语言文字工作委员会官方文本，经数字化整理后包含以下字段：

| 字段 | 类型 | 说明 |
|:---|:---:|:---|
| `id` | string | 序号，4位补零（0001–8105） |
| `char` | string | 汉字本体（UTF-8） |
| `pinyin` | string | 标准拼音（带声调）；多音字以 `|` 分隔 |
| `initial` | string | 首字母（大写），便于按音序检索 |
| `strokes` | int | 楷书笔画数 |
| `level` | int | 字表级别：1 / 2 / 3 |
| `polyphonic` | bool | 是否为多音字 |

## 四、应用场景

- 📚 **出版与印刷**：字符合规性校验，杜绝生僻字流入正式出版物
- 🖥️ **信息系统**：字库选型依据，输入法词库、字体文件最小化设计
- 🎓 **教育领域**：语文教材编排、识字率测评、义务教育识字标准
- 🔍 **NLP 预处理**：词法分析时过滤非规范字，提升语料质量
- 🌐 **国际中文教育**：汉语水平考试（HSK）、海外华文教材编制
